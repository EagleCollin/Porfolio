D&D Character Storage

The Purpose

I started this application for one main reason: I didn't like being restricted to how many characters I could store at one time on another web application.
This led me down the rabbit hole of "How in depth do I want to make this application?" That discovery process brought me to a singular conclusion:
I want this application to simplify the process for any wanna be Dungeons and Dragons nerd.


Functionality

The main sections of the application are the authorization, the api calls, and the actual display of the information.
